# Breakout Game

## Overview

Breakout Game is a browser-based arcade game developed using HTML, CSS, JavaScript, and the Canvas API. Players control a paddle to bounce a ball and destroy bricks while preventing the ball from falling below the screen.

## Features

- Ball physics
- Brick collision detection
- Paddle controls
- Keyboard input
- Smooth animations
- Classic gameplay

## Technologies Used

- HTML
- CSS
- JavaScript
- Canvas API

## Run Project

Open index.html in a browser.


## Controls

- Left Arrow → Move Left
- Right Arrow → Move Right
